Hamood | Discord Bot Python, MongoDB, AWS, JSON, HTML/CSS, Too many APIs

Hamood is a versatile Discord bot written in python that provides a variety of functions within discord servers. I intially created Hamood as an inside joke between friends, but it quickly grew to be one of my largest projects, and my first real introduction to computer science. My bot mainly provides entertainment by allowing users to custom generate their own memes and fetch reddit posts, however it also helps students with their chemistry, math, and computer science homework.

Ephemeral Chat | WebSocket Chat App JavaScript, Python, HTML/CSS, JSON, tailwindcss

A WebSocket chat app that allows users to create temporary profiles and group chats that are deleted when left.

This project was an outlet to learn more about web development and websockets, and I had a lot of fun working on it. The idea was to create a chat app similar to discord that functions like omegle. The front end was made with tailwindcss and jinja templating, while the backend server is written in python using FastApi. The client side JavaScript establishes the websocket connection and handles the data being sent and recieved from and to the user.

DiscordSubHub | Web API Python, MongoDB, Heroku, JSON, HTML/CSS

DiscordSubHub is a Feed API that subscribes Discord Webhooks to Youtube Channel updates. It works by subscribing itself to the youtube channel, through Google's Pubsubhubbub API, it then keeps tracks of the subscribed channels and their according discord webhooks to send out channel updates directly to discord.

I created DiscordSubHub for my bot Hamood to use, allowing users to subscribe to youtube channels directly from the bot. Anyone can use it through the API or directly from the site.

PIL-Stacks | Python Module Python, JSON

PIL-Stacks is a python based image template module that simplifies the process of editing images with varying external media or text. It is essentially a wrapper for the Pillow Python Imaging Library and handles everything to do with the image generation process, leaving only custimization up to the user.

The module also features a gui editor built with Pygame that lets you quickly create templates as well as use previews to see how they would look.

I currently use this module to power the image generation of my discord bot.

discomoji.js | JavaScript Library JavaScript

Discomoji.js is a tiny library that allows the use of Custom Discord Emojis such as [a:speen:847583231850446858] or [a:blobDance:846930212251107330] on a site. It works by finding discord emojis tags (ex. a:speen:847583231850446858) and replacing them with their emoji image, inline and sized with the text.

I created discord.js while creating this site so that I could use discord emojis on it.
